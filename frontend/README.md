# ZENVY Frontend

AI-first ecommerce frontend for the ZENVY backend.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Zustand
- Axios

## Run

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env.local` if you want to override the backend URL.

Default backend URL:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080
```
